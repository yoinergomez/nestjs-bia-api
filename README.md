# Nestjs Bia API

## Installation

```bash
# node version >=18.12.1
$ npm install
```

## Running the app

```bash
# start postgres db
$ docker-compose up -d

# start app
$ npm run start

```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
